Method-Tec - Redirecionamento por URL

O que é redirecionar para um elemento?

Redirecionar o usuário para um elemento específico em uma página HTML significa levá-lo diretamente para uma seção ou conteúdo dentro dessa página. Isso é feito usando identificadores (id) e a URL com uma hash (#).

Exemplo Prático

Considere o seguinte código HTML:

<div id="card2"> <h2>Card do Segundo Método</h2> <p>Conteúdo do segundo método.</p> </div>

Para redirecionar o usuário diretamente para esse elemento, basta usar uma URL como esta:

https://method-tec.vercel.app/aula3.html#p

Passo a Passo

  1. Adicione o atributo id ao elemento que você deseja redirecionar. No exemplo, o id é card2.
  2. Inclua o #id ao final da URL da página, como em #card2.
  3. Compartilhe a URL com o usuário.
