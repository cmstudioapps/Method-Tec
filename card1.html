Como Criar um App do Site com PWA

← Voltar

O que é um PWA?

Um **Progressive Web App (PWA)** é uma tecnologia que permite transformar sites em apps instaláveis no celular ou computador. Ele funciona como um app nativo, mas sem precisar de lojas como Google Play ou App Store.

Passo a Passo para Criar um PWA

  1. Crie um arquivo manifest.json: Esse arquivo informa ao navegador os detalhes do app. Exemplo: 
    {
  "name": "Method-Tec",
  "short_name": "Method-Tec",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#121212",
  "theme_color": "#ff0077",
  "icons": [
    {
      "src": "logo192.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "logo512.png",
      "sizes": "512x512",
      "type": "image/png"
    }
  ]
}
  2. Adicione o manifest.json no HTML: Inclua a seguinte linha dentro da tag ``: 
    <link rel="manifest" href="manifest.json">
  3. Implemente um Service Worker: Um Service Worker é um script que permite o funcionamento offline. Crie um arquivo chamado `sw.js`: 
    self.addEventListener('install', (event) => {
  console.log('Service Worker Instalado');
  event.waitUntil(
    caches.open('method-tec-cache').then((cache) => {
      return cache.addAll([
        '/',
        '/index.html',
        '/style.css',
        '/logo192.png'
      ]);
    })
  );
});

self.addEventListener('fetch', (event) => {
  event.respondWith(
    caches.match(event.request).then((response) => {
      return response || fetch(event.request);
    })
  );
});
  4. Registre o Service Worker no JavaScript: No arquivo `main.js` ou em um script do site, adicione: 
    if ('serviceWorker' in navigator) {
  navigator.serviceWorker.register('/sw.js').then(() => {
    console.log('Service Worker Registrado!');
  });
}
  5. Teste o PWA: Abra o site no navegador, e:
    • No Chrome, vá em Configurações > Instalar App.
    • Ou no celular, veja o botão Adicionar à tela inicial.

Exemplo ao Vivo

1. Abra o site: Visite o Method-Tec no navegador.
2. Clique no botão: Escolha "Adicionar à Tela Inicial" ou "Instalar App".
3. Aproveite: O site agora está instalado como um app!

Benefícios do PWA

  • Funciona offline.
  • Carregamento rápido.
  • Não depende de lojas de apps.
  • Fácil instalação e desinstalação.
